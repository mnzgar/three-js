## 🚀 Prepare the development environment

1.  **Clone the repository**

    ```zsh
    git@github.com:mnzgar/three-js.git
    ```

2.  **Run**

    Navigate to the directory and run it.

    ```zsh
    cd three-js/
    npm i
    npm run dev
    ```
